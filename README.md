# influencer-dashboard-project
This repository contains influencer dataset analysis including synopsis, project report, Excel dashboard, and AI chatbot agent.
## Influencer Marketing Analysis Project

##  Project Overview

This project focuses on analyzing influencer marketing data to understand audience behavior, platform preference, and decision-making factors. It also includes a chatbot agent to assist with content generation and insights.

## Project Contents

###  1. Synopsis

* Introduction to influencer marketing
* Problem statement
* Objectives of the study
* Scope of the project

### 2. Project Report

* Detailed explanation of methodology
* Data analysis and findings
* Charts and interpretations
* Conclusion and future scope

### 3. Dataset

* Cleaned influencer dataset
* Structured for analysis
* Includes audience responses and platform data

### 4. Dashboard (Excel)

* Interactive dashboard using Pivot Tables & Charts
* Key insights:

  * 44.7% negative audience response
  * Instagram is most preferred platform (51.7%)
  * Data distribution across segments

### 5. Chatbot Agent

* AI-based chatbot for marketing content generation
* Helps in:

  * Writing ads
  * Generating captions
  * Providing suggestions

## Tools & Technologies Used

* Microsoft Excel
* GitHub
* ChatGPT / AI tools

## Objectives

* Analyze audience behavior
* Identify best social media platforms
* Evaluate influencer impact
* Build an AI chatbot for automation

## Conclusion

The analysis shows that audience engagement is mostly negative, while Instagram remains the dominant platform. The chatbot enhances usability by providing automated content solutions.
